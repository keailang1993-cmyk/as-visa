# AS VISA Sprint 8 Dynamic Mission Flow

## Goal

After each document is completed, the Mission page automatically switches to the next single task.

## Flow

1. Upload Passport
2. Upload ID Card
3. Upload Bank Statement
4. Mission Completed

## Rule

The Mission Card displays only one current task at any time.

## Current State

Mission progress is stored locally in `localStorage` while the app has no backend mission service.
