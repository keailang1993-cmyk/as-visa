# AS VISA Sprint 7 AI Review Screen

## Goal

Immediately after document upload, show AI reviewing the uploaded passport.

## Duration

The review sequence lasts about 4 seconds.

## States

1. Reviewing...
2. Passport detected.
3. Checking validity...
4. Checking completeness...
5. Review Complete.

## Animation

Each state uses a smooth minimal fade.

## Completion

After review completion, the screen automatically navigates to the next Mission at `/mission-next`.

No button is required on the AI Review screen.
