# AS VISA Sprint 6 Upload Passport

## Goal

Build the upload step for the passport mission.

## Flow

1. Mission page CTA routes to `/upload-passport`.
2. User chooses Camera or Photo Upload.
3. User previews the selected passport file.
4. User can retry.
5. User confirms success.
6. Successful upload routes to `/ai-review`.

## Constraints

- Uses NOIR Design System upload components.
- Does not build dashboard, sidebar, or extra navigation.
- Does not implement backend storage yet.
