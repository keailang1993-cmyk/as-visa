# NOIR Playground

The NOIR Playground is the design-system review page for AS VISA.

Route:

```text
/playground
```

Purpose:

- Display every reusable component in one place.
- Show design token behavior for colors, typography, spacing, radius, shadows, and motion.
- Give designers, developers, and future AI agents one reference surface before building product pages.

Rules:

- The playground is not a business page.
- It does not implement login, dashboard, or visa workflow behavior.
- It uses reusable `@as-visa/ui` components and NOIR design tokens only.
