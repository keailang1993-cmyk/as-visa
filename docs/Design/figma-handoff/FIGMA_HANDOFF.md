## Design Tokens

Colors:
- Background: `#FFFFFF`
- Foreground: `#111111`
- Muted Text: `#6B6B6B`
- Border: `#E5E5E7`
- Surface: `#F7F7F7`
- Soft Surface: `#F8F8F8`

Typography:
- Title: `28px / 600`
- Section Title: `20px / 600`
- Body: `16px / 400`
- Description: `14px / 400`
- Caption: `12px / 500`

Spacing:
- `8px` grid
- Main page horizontal padding: `24px`
- Section gap: `24px / 32px`
- Card padding: `24px`

Radius:
- Card radius: `16px`
- Button radius: `16px`
- Input radius: `16px`
- App shell radius: `32px` when shown in preview frame

Buttons:
- Primary height: `48px`
- Secondary height: `48px`
- One primary CTA per page

Motion:
- `120ms` to `180ms`
- Opacity
- Small scale
- No fancy animation

## Rules

- Black / white / gray only
- No gradients
- No colorful UI
- No heavy shadows
- No dashboard feeling
- No random spacing
- Every page must have one clear focus
- User should instantly know what to do next
