# AS VISA NOIR Design System

NOIR is the reusable design system for AS VISA. It is restrained: black, white, gray, thin borders, soft radius, calm spacing, and typography-led hierarchy.

## Foundations

- Colors: black, white, and neutral gray only.
- Typography: system-first sans stack.
- Spacing: tokenized spacing scale.
- Radius: soft, consistent radius tokens.
- Shadow: subtle neutral elevation only.
- Motion: short, calm transitions.
- Grid: page, content, narrow, and gap tokens.
- Z-index: base, dropdown, sticky, overlay, modal, toast.
- Opacity: disabled, muted, overlay, visible.

## Component Groups

- Buttons: primary, secondary, ghost, text, loading, icon.
- Inputs: text, phone, OTP, search, textarea, country selector, date picker.
- Cards: mission, upload, AI, progress, document, notification, status, profile.
- Upload: upload area, queue, camera guide, preview, OCR result, retry.
- Progress: timeline, mission progress, status badge, step indicator, completion card.
- AI: assistant, tip, suggestion, warning, summary, thinking.
- Feedback: loading, skeleton, empty, error, success, toast, dialog, bottom sheet.
- Navigation: header, back button, bottom navigation, profile menu.

## Rule

Future pages should assemble screens from `@as-visa/ui` instead of creating page-specific primitives.
